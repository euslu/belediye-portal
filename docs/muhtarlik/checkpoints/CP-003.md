# CP-003 — Mahalle Detay + Parametrik Tablo + Sidebar Düzenlemesi
Tarih: 28.03.2026

## Tamamlananlar

### Mahalle Detay Sayfası (/muhtarlik/mahalle/:ilce/:mahalle)
- Muhtar bilgisi kartı (ad, nüfus, kırsal/merkez)
- Başvuru özeti kartı (toplam, tamamlandı, devam, tamamlanmadı, ort. süre)
- Başvurular sekmesi: durum filtreli tablo, satırdan düzenleme modalı
- Yatırımlar sekmesi: JSON'dan kronolojik liste, kalem ikonu ile düzenleme

### Yatırım Düzenleme
- PUT /api/muhtarbis/yatirim/:ilce/:mahalle/:index endpoint
- Alanlar: açıklama, tahmini bedel, daire, durum, cevap
- MuhtarbisEditLog tablosuna log kaydı

### Parametrik Sütunlar
- YATIRIM_SUTUNLARI ve BASVURU_SUTUNLARI config objesi
- localStorage'da kullanıcı tercihi saklama
  keys: muhtarlik_yatirim_sutunlar / muhtarlik_basvuru_sutunlar
- Sütun görünürlüğü toggle — anında etkili

### Sidebar
- MUHTARLıKLAR grubu: GÖREVLERİM gibi statik, accordion yok
- Alt linkler: Başvuru Takibi / Muhtarlar / Raporlar / Ayarlar

### Muhtarlık Ayarlar Sayfası (/muhtarlik/ayarlar)
- Yatırım ve başvuru tablosu sütun yönetimi
- localStorage sync
